# Ramadan Tracker

A web application to track good deeds during Ramadan, including prayers, duas, and Quran recitation. The app helps users monitor their progress and stay motivated throughout the month.

## Features
- Track daily prayers
- Log duas and supplications
- Record Quran recitation progress
- View insights and progress reports

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/ramadan-tracker.git
   ```
2. Navigate to the project folder:
   ```sh
   cd ramadan-tracker
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used
- React.js (with Vite)
- Zustand (for state management)
- Tailwind CSS (for styling)
- React Query (for data fetching)
